## Linguagens de Programação e Áreas de Interesse

Ao longo da minha formação, tive contato com diferentes linguagens de programação e tecnologias, o que me permitiu desenvolver uma visão mais ampla sobre o desenvolvimento de software. As linguagens com as quais possuo maior afinidade são **Java** e **Python**, sendo o Java a linguagem que mais utilizo atualmente para aprimorar minha lógica de programação, desenvolver aplicações orientadas a objetos e explorar conceitos de arquitetura de software e desenvolvimento de APIs. O Python também faz parte da minha rotina de estudos, principalmente pela sua versatilidade em automação, manipulação de dados e resolução de problemas.

Além do desenvolvimento back-end, também tenho interesse pelo desenvolvimento web. Gosto de trabalhar com a tríade **HTML, CSS e JavaScript**, que considero a base para a construção de interfaces modernas e acessíveis. Durante meus estudos, tive a oportunidade de desenvolver pequenos projetos utilizando **React** e **React Native**, adquirindo uma compreensão inicial sobre o desenvolvimento de aplicações web e mobile. Embora essas tecnologias não sejam o meu foco atual, elas contribuíram para ampliar minha visão sobre o ecossistema do desenvolvimento de software.

Atualmente, meu principal objetivo é fortalecer os conhecimentos em **Linux**, administração de sistemas e infraestrutura, áreas que considero fundamentais para a construção de aplicações robustas e escaláveis. Paralelamente, continuo aperfeiçoando minha lógica de programação e minhas habilidades em **Java** e **Python** por meio de estudos diários, resolução de desafios de programação e desenvolvimento de projetos práticos. Acredito que a combinação entre uma base sólida em programação e conhecimentos de infraestrutura é essencial para formar um profissional capaz de compreender todo o ciclo de desenvolvimento e implantação de sistemas.
