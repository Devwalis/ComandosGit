Ao iniciar minha trajetória na área de tecnologia, comecei consolidando meus conhecimentos por meio do curso Técnico em Desenvolvimento de Sistemas, que me proporcionou uma base sólida em programação, lógica computacional e desenvolvimento de aplicações. Desde então, adotei uma postura proativa em relação ao aprendizado, buscando constantemente conteúdos além da grade curricular, realizando cursos complementares, estudando por conta própria e praticando diariamente para aperfeiçoar minhas habilidades técnicas. Ao longo da minha formação, busquei complementar o aprendizado acadêmico com cursos voltados ao desenvolvimento de software e infraestrutura. Concluí o curso AWS re/Start, com 420 horas de formação, onde desenvolvi conhecimentos em computação em nuvem, Linux, redes, segurança, Python e serviços da AWS. Também realizei cursos de Lógica de Programação, HTML, CSS, JavaScript, Bootstrap, SQL e Banco de Dados, fortalecendo ainda mais minha base técnica. Durante esse período, desenvolvi diversos projetos para consolidar os conhecimentos adquiridos, incluindo uma plataforma de e-commerce baseada em microsserviços utilizando Java e Spring Boot, um sistema de cardápio online, aplicações para estudo de algoritmos e estruturas de dados, além de exercícios práticos publicados no GitHub. Cada projeto foi desenvolvido com foco na aplicação de boas práticas de programação, versionamento com Git, modelagem de banco de dados e desenvolvimento de APIs REST, contribuindo para a construção de um portfólio sólido e demonstrando meu compromisso com a aprendizagem contínua e com a evolução profissional na área de tecnologia.
